TM1638 Timer
============

Hardware
--------

TM1638 with:
* CLK on pin 9
* DIO on pin 8
* STB0 on pin 7

Buzzer on pin 12

Setting the timer on the Arduino
--------------------------------

Buttons, left to right:
0. Change intensity
1. Increment hours
2. Decrement hours
3. Increment minutes
4. Decrement minutes
5. Increment seconds
6. Decrement seconds
7. Start
